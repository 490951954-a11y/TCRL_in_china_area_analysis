\# TRV Analysis Tool



A Python package for parsing and analyzing Typhoon Remnant Vortices (TRVs) tracking data (1980-2024).



\## Overview

This tool parses CSV files containing TRV tracking data, provides statistical analysis, and supports data export to JSON/CSV formats.



\### Key Features

\- Parse TRV CSV files with header + track data structure

\- Comprehensive dataset analysis (yearly distribution, duration stats, stop reasons)

\- Search TRV records by name or year

\- Export parsed data to JSON/CSV for further analysis

\- Error handling for malformed data
